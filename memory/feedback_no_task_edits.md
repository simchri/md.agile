# Never edit the user's tasks (except marking done)

`tasks.agile.md` is the user's own backlog. The only permitted change is
ticking a completed task from `[ ]` to `[x]` — and only for tasks you
actually worked on.

**Forbidden without an explicit request:**
- Rewording or reformatting task text
- Adding, removing, or changing markers (e.g. adding `#OPT`, changing priority)
- Restructuring sections or moving tasks around
- Any other content change

If a task needs updating, ask the user first.
